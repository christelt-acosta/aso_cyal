Práctica 1 - Introducción a Vagrant

Pasos que seguí

1. Descargar Vagrant

Descargar e instalar Vagrant desde su sitio oficial.

2. Verificar la lista de boxes

Ejecutar el siguiente comando para ver qué boxes están instaladas:

vagrant box list

Como no hay ninguna, buscamos una en Vagrant Boxes. En este caso, elegimos peru/my_ubuntu-20.04-server-amd64.

3. Descargar el box

Para descargar el box seleccionado:

vagrant box add peru/my_ubuntu-20.04-server-amd64

4. Crear Vagrantfile

Primero, creamos una carpeta para la máquina virtual, luego ejecutamos:

vagrant init -m

Esto generará el archivo Vagrantfile.

5. Configurar el Vagrantfile

Ejemplo de configuración del archivo Vagrantfile:

# -*- mode: ruby -*-
# vi: set ft=ruby :

Vagrant.configure("2") do |config|
    config.vm.box = "peru/my_ubuntu-20.04-server-amd64"
    config.vm.hostname = "servercyal"
    config.vm.synced_folder "./carpetadata", "/data", create=true
    config.vm.provider "virtualbox" do |vb|
        vb.name = "ubuntu server"
        vb.memory = 2048
        vb.cpus = 2
    end
end

6. Iniciar la máquina

Para levantar la máquina, ejecutar:

vagrant up

Es importante estar en la carpeta donde está el Vagrantfile al ejecutar este comando.

7. ¡Máquina creada!

Con esto, la máquina virtual está lista para usarse.
